XP (Experience Points) is earned by completing quests and using the training timer. Each quest has a base XP value, but you can earn bonus XP through consistency.

  • Base XP: Set per quest (typically 10-30 XP)
  • Daily Momentum: +50% XP for your first quest each day
  • Comeback Bonus: +15 XP if you missed yesterday but returned today
  • Perfect Essential Day: +20 XP when all essential quests are completed

Maintaining daily consistency unlocks powerful XP multipliers:

  • 3-6 days: +10% bonus XP on all quests
  • 7-13 days: +25% bonus XP on all quests
  • 14-29 days: +50% bonus XP on all quests
  • 30+ days: +75% bonus XP on all quests

ğŸ’¡ Tip: A 30-day streak can turn a 20 XP quest into 35 XP or more with all bonuses combined!
